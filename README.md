# Asian Distributor App

Este repositorio contiene un módulo personalizado de Odoo 18 diseñado específicamente para las necesidades de una empresa importadora y distribuidora de productos asiáticos.

## Funcionalidades principales (En desarrollo)
- Trazabilidad del País de Origen
- Control de importaciones
- Procesos optimizados de Ventas, Compras e Inventario

## Arquitectura
Este módulo sigue el marco de desarrollo guiado por especificaciones (Agent Teams Lite SDD).
